## Regles d'utilisation

### Lors de la creation de contenu

1. Verifier a quel cluster le sujet appartient
2. Cibler le mot-cle du cluster ou longue traine approprie
3. Lier vers le contenu pilier du cluster
4. Inclure les mots-cles LSI naturellement
5. Lier vers les autres articles du cluster

### Lors de l'optimisation de contenu existant

1. Verifier que le contenu cible le bon mot-cle
2. Chercher une cannibalisation avec d'autres pages
3. S'assurer que le mot-cle apparait aux emplacements critiques
4. Mettre a jour si l'intention de recherche a evolue
5. Ajouter des liens internes vers/depuis les contenus du cluster
