## Usage Rules

### When Creating Content
1. Check which cluster the topic belongs to
2. Target the cluster or long-tail keyword
3. Link back to the cluster's pillar content
4. Include LSI keywords naturally
5. Link to other articles in the same cluster
6. Research keywords in all target languages before writing

### When Optimizing Existing Content
1. Verify the content targets the right keyword
2. Check for keyword cannibalization with other pages
3. Ensure keyword appears at critical placements
4. Update if search intent has evolved
5. Add internal links to/from cluster content
